In this last step, you are provided with a web script to install in the header of your webpage. This script allows you to gather detailed analytics on which answer engines are visiting your web page, and what content they are scraping. These insights are surfaced in our **Anlaytics** module.

This script is also used by the  **Boost** module to integrate content that is optimized for the answer engine directly into your website, in order to improve your metrics.
